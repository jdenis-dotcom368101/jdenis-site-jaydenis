# Site web JDENIS Consulting — jaydenis.com

Site professionnel de **Jérôme Denis**, consultant en **Intelligence Artificielle** et **transformation digitale** basé à Toulouse.

## Statut

**Production** — site en ligne sur [https://jaydenis.com](https://jaydenis.com)

## Description

Site vitrine bilingue (français / anglais) présentant l'activité de JDENIS Consulting :
- Diagnostic IA et accompagnement stratégique pour PME et ETI
- Formation entreprise (ChatGPT, Claude, Mistral, prompt engineering)
- Gouvernance data & conformité AI Act
- Conférences et interventions en écoles supérieures

## Technologies

- **HTML5** sémantique
- **CSS3** (styles intégrés, design responsive)
- **JavaScript vanilla** (sans framework)
- **JSON-LD / Schema.org** pour le référencement structuré
- **Google Fonts** (Inter, Playfair Display)
- **Optimisation SEO** : sitemap.xml, robots.txt, balises Open Graph & Twitter Card, métadonnées géographiques

Site statique — déployable sur tout hébergement web standard (Apache, Nginx, Netlify, Vercel, GitHub Pages, etc.).
